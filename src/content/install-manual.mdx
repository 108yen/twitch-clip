# アプリのインストール方法

## iOS

### Safariからインストールする場合

1. <ShareIcon />
   &nbsp;ボタンをタップ
2. ホーム画面に追加<SquarePlusIcon />&nbsp;をタップ

### Chromeからインストールする場合

1. アドレスバーの&nbsp;<ShareIcon />&nbsp;ボタンをタップ
2. ホーム画面に追加<SquarePlusIcon />&nbsp;をタップ

## Android

### Chromeからインストールする場合

1. <EllipsisVerticalIcon />
   &nbsp;ボタンをタップ
2. ホーム画面に追加&nbsp;をタップ
3. インストール
